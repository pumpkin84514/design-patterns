Saga 长事务

Saga

订外卖：下单→付款→商家接单→配送。如果商家不接单，退款取消订单。

🔑 记忆：Saga = 外卖流程 → 每一步有补偿 → 失败就回退 → 长事务拆分
🎯 电商订单、旅游预订、金融转账、长流程业务 点击学习
🔴 核心 架构设计层
📜

事件溯源

Event Sourcing

银行账本：不记录余额，只记录每笔存取，余额是算出来的。

🔑 记忆：事件溯源 = 银行账本 → 记录事件 → 状态由事件推导 → 可回溯
🎯 审计追踪、状态重建、复杂业务、时间旅行 点击学习
🔴 核心 架构设计层
⚡

CQRS 读写分离

Command Query Separation

餐厅：厨师负责做菜（写），服务员负责上菜（读），分工明确。

🔑 记忆：CQRS = 读写分离 → 命令和查询分开 → 各自优化 → 不互相影响
🎯 高并发读写、复杂查询、事件溯源、报表系统 点击学习
🔴 核心 架构设计层
🚪

API 网关

API Gateway

酒店前台：客人来了先找前台，前台再分配房间、处理登记、解答问题。

🔑 记忆：API网关 = 酒店前台 → 统一入口 → 过滤/路由/聚合 → 不直接找服务员
🎯 微服务入口、统一认证、协议转换、流量聚合 点击学习
架构设计层

事件总线

Event Bus

公司公告栏：有人发布消息，所有人都能看见，不用一个个通知。

🔑 记忆：事件总线 = 公告栏 → 发布订阅 → 松耦合 → 广播通知
🎯 服务间通信、状态变更通知、领域事件传播 点击学习
架构设计层

服务独享数据库

Database per Service

每家公司都有自己的账本，互不干涉。你要查账只能找对方公司。

🔑 记忆：独享数据库 = 各自账本 → 不共享数据 → 通过API通信 → 独立演进
🎯 微服务拆分、数据自治、团队独立、技术异构 点击学习
架构设计层
🔪

数据分片

Sharding

图书馆按字母分书：A-F放一楼，G-L放二楼，不用在一层楼找所有书。

🔑 记忆：数据分片 = 图书馆分区 → 按规则拆分 → 均匀分布 → 分散压力
🎯 大表拆分、按用户/地域分片、水平扩展、分布式数据库 点击学习
架构设计层
🥪

分层缓存

Cache Hierarchy

办公室：桌上→抽屉→文件柜，常用文件放最近处。

🔑 记忆：分层缓存 = 桌面/抽屉/文件柜 → 越近越快 → 命中率高 → 分层命中
🎯 L1/L2/L3缓存、Redis/本地缓存/CDN、热点数据加速 点击学习
🔴 核心 代码实现层
🔌

熔断器

Circuit Breaker

电路保险丝：电流过载时自动断开，防止烧毁电器。

🔑 记忆：熔断器 = 保险丝 → 故障过多自动断开 → 保护系统不被拖垮
🎯 微服务调用、第三方API、数据库连接、远程服务 点击学习
🔴 核心 代码实现层
🚦

限流

Rate Limiting

高速收费站：每秒只放行固定数量车辆，防止拥堵。

🔑 记忆：限流 = 收费站 → 控制流速 → 超过就排队或拒绝 → 保护系统
🎯 API网关、接口防刷、抢购活动、验证码发送 点击学习
🔴 核心 代码实现层
🔄

重试

Retry

打电话对方占线，过一会儿再打，不是永远等下去。

🔑 记忆：重试 = 占线再打 → 不是立即放弃 → 有策略地重试
🎯 网络请求、消息发送、数据库写入、分布式锁获取 点击学习
🔴 核心 代码实现层
⏱️

超时

Timeout

微波炉定时：设10分钟，到时间就自动停止，不会无限加热。

🔑 记忆：超时 = 微波炉定时 → 到点就停 → 防止无限等待
🎯 RPC调用、HTTP请求、数据库查询、锁等待 点击学习
🔴 核心 代码实现层
🛳️

舱壁隔离

Bulkhead

轮船防水舱：一个舱进水，其他舱不受影响，船不会沉。

🔑 记忆：舱壁隔离 = 防水舱 → 故障隔离 → 互不影响 → 局部问题
🎯 线程池隔离、服务分组、资源分区、数据库分库 点击学习
代码实现层
🔽

降级

Degradation

医院急诊：病人太多时，优先救治危重病人，轻症暂缓。

🔑 记忆：降级 = 急诊分诊 → 保核心 → 非核心暂停 → 有损服务
🎯 大促活动、服务过载、依赖故障、非核心功能关闭 点击学习
🔴 核心 代码实现层
♻️

幂等

Idempotency

电梯按钮：按一次和按十次效果一样，电梯只来一趟。

🔑 记忆：幂等 = 电梯按钮 → 重复操作结果一样 → 按多少次都一样
🎯 支付回调、接口重试、消息消费、表单提交 点击学习
代码实现层
🔐

分布式锁

Distributed Lock

公共厕所：有人进去就锁门，外面的人必须等待。

🔑 记忆：分布式锁 = 厕所锁门 → 互斥访问 → 防止争抢 → 排队
🎯 库存扣减、订单防重、定时任务、全局唯一ID 点击学习
🔴 核心 运维保障层
🔍

服务发现

Service Discovery

医院导诊台：新医生来了登记，病人来了问导诊台找哪个医生。

🔑 记忆：服务发现 = 导诊台 → 注册/查询 → 动态发现 → 不用硬编码地址
🎯 微服务注册、动态扩缩容、K8s服务、负载均衡前置 点击学习
🔴 核心 运维保障层

负载均衡

Load Balancing

超市收银台：多个柜台分流顾客，避免某个柜台排太长队。

🔑 记忆：负载均衡 = 多收银台 → 分流排队 → 均匀分配 → 不扎堆
🎯 网关分发、服务集群、数据库读写分离、CDN调度 点击学习
🔴 核心 运维保障层
💓

健康探针

Health Probe

体检中心：定期给人做检查，有问题早发现早治疗。

🔑 记忆：健康探针 = 体检 → 定期检测 → 不健康剔除 → 自动恢复
🎯 K8s存活检查、服务状态监控、自动重启、流量摘除 点击学习
运维保障层

调和回路

Reconciliation Loop

扫地机器人：设定目标（全扫一遍），不断调整路径直到完成。

🔑 记忆：调和回路 = 扫地机器人 → 实际vs期望 → 持续修正 → 自动对齐
🎯 K8s控制器、配置漂移修复、资源状态同步、期望管理 点击学习
运维保障层

金丝雀发布

Canary Release

煤矿工人带金丝雀下矿：先放小鸟探路，安全了人再下去。

🔑 记忆：金丝雀 = 先派鸟探路 → 小流量验证 → 没问题再全量 → 风险控制
🎯 版本发布、A/B测试、灰度验证、新功能上线 点击学习

GitOps

GitOps

装修图纸：设计师画好图纸，工人按图施工，装修完跟图纸一致。

🔑 记忆：GitOps = 装修图纸 → Git是唯一真相 → 自动部署 → 状态一致
🎯 K8s部署、基础设施管理、配置即代码、自动化运维 点击学习

分布式追踪

Distributed Tracing

快递跟踪：从发货到签收，每个中转站都扫码记录，全程可追踪。

🔑 记忆：分布式追踪 = 快递跟踪 → 每个节点记录 → 串联全链路 → 定位问题
🎯 性能分析、故障定位、调用链分析、延迟排查 点击学习
运维保障层

结构化日志

Structured Logging

表格登记：不是手写笔记，而是统一表格，方便分类查找。

🔑 记忆：结构化日志 = 统一表格 → 标准化格式 → 可检索分析 → 日志即数据
🎯 日志聚合、ELK/Loki、异常分析、业务监控 点击学习
运维保障层
📜

声明式配置

Declarative Config

点外卖：你说"我要一份宫保鸡丁"，不用管厨房怎么炒。

🔑 记忆：声明式配置 = 点外卖 → 描述期望 → 系统自动达成 → 不用关心过程
🎯 K8s YAML、Terraform、基础设施编排、自动化部署 点击学习
