单例模式

Singleton

一个国家只能有一个总统，不管多少人找他，都是同一个人。

🔑 记忆：名字里有"单" → 只有一个 → 总统唯一 → 全局共享
🎯 数据库连接池、配置管理、日志对象、线程池 点击学习
创建型
🏭

工厂方法

Factory Method

开餐厅：不同厨师做不同菜品，但统一按"厨师做餐"流程。

🔑 记忆：工厂方法 = 子类决定造什么 → 父类定流程 → 子类填工厂
🎯 UI组件创建、支付渠道选择、游戏角色生成 点击学习
创建型
🏭🏭

抽象工厂

Abstract Factory

苹果工厂产iPhone+iPad，华为工厂产Mate+MatePad，各产一套。

🔑 记忆："抽象" = 一组工厂 → 不只生产一个 → 生产一整套配套产品
🎯 跨平台UI套件、数据库切换、主题系统 点击学习
创建型
🔧

建造者模式

Builder

组装电脑：选CPU、选显卡、选内存，一步一步配出来。

🔑 记忆：建造者 = 一步一步搭建 → 像搭积木 → 每个步骤可自定义
🎯 复杂对象配置、SQL构造器、测试数据生成 点击学习
创建型
📋

原型模式

Prototype

用复印机复印文件，得到一份一模一样的新文件。

🔑 记忆：原型 = 复制一份 → 复印机 → 不用从头写 → 改改就能用
🎯 对象克隆、缓存对象复制、复杂对象初始化成本高 点击学习
结构型
🔌

适配器模式

Adapter

出国旅行带转换插头，让中国充电器在外国插座上使用。

🔑 记忆：适配器 = 转换插头 → 不兼容的东西 → 中间加适配层 → 就能用了
🎯 旧系统兼容、第三方API封装、数据格式转换 点击学习
结构型
🎀

装饰器模式

Decorator

买一杯咖啡，可以加奶、加糖、加珍珠，每层不改变原来咖啡。

🔑 记忆：装饰器 = 包装礼物 → 一层一层包 → 每层加新功能 → 核心不变
🎯 HTTP中间件、权限层层叠加、输入验证链 点击学习
结构型
🕵️

代理模式

Proxy

明星请经纪人，粉丝找经纪人预约，经纪人过滤不合适的请求。

🔑 记忆：代理 = 经纪人/代理 → 代替本人做事 → 过滤、控制、增强
🎯 权限控制、懒加载、缓存代理、日志监控 点击学习
结构型
🎛️

外观模式

Facade

汽车"一键启动"按钮，隐藏引擎、电路、油路的复杂启动。

🔑 记忆：外观 = 一键启动 → 隐藏内部复杂 → 只暴露一个简单按钮
🎯 复杂API封装、SDK简化、启动流程、多步骤聚合 点击学习
结构型
🌉

桥接模式

Bridge

遥控器和电视：可以换遥控器，也可以换电视，两者独立变化。

🔑 记忆：桥 = 连接两岸 → 遥控器和电视是独立维度 → 可随意组合
🎯 跨平台渲染、驱动切换、UI主题与控件分离 点击学习
结构型
🌳

组合模式

Composite

文件夹里可以放文件，也可以放子文件夹，子文件夹里还能继续放。

🔑 记忆：组合 = 树 → 文件+文件夹 → 统一对待 → 递归结构
🎯 UI组件树、菜单系统、组织架构、文件系统 点击学习
结构型
♻️

享元模式

Flyweight

共享单车：一百个人骑，但车只有100辆，大家共享。

🔑 记忆：享元 = 共享单元 → 共享单车 → 共享对象 → 池化复用
🎯 游戏角色池、字符串常量池、粒子系统、地图瓦片缓存 点击学习
行为型
🎯

策略模式

Strategy

去机场：可以打车、坐地铁、骑车，目的相同，方式不同。

🔑 记忆：策略 = 多种方案选一 → 去机场方式不同 → 目的相同，策略不同
🎯 支付方式选择、排序算法切换、压缩策略、验证规则 点击学习
行为型
👀

观察者模式

Observer

关注公众号后，公众号发文章，所有粉丝自动收到推送。

🔑 记忆：观察 = 关注公众号 → 一有更新 → 自动推送 → 被动接收
🎯 事件系统、消息队列、数据绑定、直播间推送、股票行情 点击学习
行为型
📋

模板方法

Template Method

泡茶流程：烧水→放茶叶→倒水→等待。绿茶红茶步骤一样，茶叶不同。

🔑 记忆：模板 = 固定流程 → 骨架不变 → 某些步骤子类填 → 泡茶流程
🎯 框架设计、测试用例模板、数据处理流程、爬虫模板 点击学习
行为型
🎮

命令模式

Command

遥控器按钮：每个按钮封装一个命令，按下去就执行。

🔑 记忆：命令 = 遥控器按钮 → 封装一个动作 → 可执行、可撤销、可记录
🎯 撤销重做、任务队列、遥控器、宏命令、事务系统 点击学习
行为型
🔁

状态模式

State

手机：响铃模式→静音模式→振动模式，按电源键切换。

🔑 记忆：状态 = 不同状态不同行为 → 手机模式切换 → 状态变了，行为跟着变
🎯 订单状态流转、游戏角色状态、播放器状态、网络连接状态 点击学习
行为型
⛓️

职责链

Chain of Responsibility

请假审批：组长批3天→经理批7天→总监批15天→更长找CEO。

🔑 记忆：职责链 = 请假审批链 → 一级一级往上走 → 直到有人处理 → 责任链
🎯 审批流程、日志过滤链、异常处理链、中间件、权限校验链 点击学习
行为型
🏠

中介者模式

Mediator

房产中介：买卖双方不直接联系，都通过中介沟通，避免混乱。

🔑 记忆：中介者 = 房产中介 → 所有人找中介 → 中介统一协调 → 降低混乱
🎯 聊天室、MVC控制器、协调多个组件、UI事件总线 点击学习
行为型
📻

迭代器模式

Iterator

电视遥控器：按"下一个"就切下一台，不用知道电视台怎么排列。

🔑 记忆：迭代器 = 遥控器下一台 → 不关心内部结构 → 只管按顺序访问
🎯 遍历集合、分页器、树形遍历、数据库游标、文件读取 点击学习
行为型
🩺

访问者模式

Visitor

医生查房：对不同病人（老人、小孩、孕妇）做不同检查。

🔑 记忆：访问者 = 医生查房 → 同一操作（体检）→ 不同对象不同处理 → 不改变病人
🎯 编译器AST遍历、报表生成、复杂对象结构操作、数据导出 点击学习
行为型
💾

备忘录模式

Memento

游戏存档：打不过BOSS就读档，回到之前状态重新挑战。

🔑 记忆：备忘录 = 游戏存档 → 保存状态 → 随时恢复 → 快照
🎯 撤销重做、游戏存档、版本回滚、事务快照、浏览器历史 点击学习
行为型
🗣️

解释器模式

Interpreter

翻译器：把"你好"翻译成英文"Hello"，把规则转换成可执行动作。

🔑 记忆：解释器 = 翻译 → 把规则文本变成可执行代码 → 如正则表达式引擎
🎯 简单规则引擎、SQL解析、表达式求值、配置文件解析 点击学习
